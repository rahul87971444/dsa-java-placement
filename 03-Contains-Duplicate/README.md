# 🔥 Contains Duplicate | LeetCode 217

One of the easiest HashSet interview questions.

---

# 📌 Problem Statement

Given an integer array, determine whether any value appears at least twice.

Return

```
true
```

if duplicate exists.

Otherwise

```
false
```

---

## Example

Input

```
[1,2,3,1]
```

Output

```
true
```

---

Input

```
[1,2,3,4]
```

Output

```
false
```

---

# 🎯 Learning Objectives

- HashSet

- Array Traversal

---

# 🚀 Approaches

| Approach | Time | Space |
|----------|------|-------|
| Brute Force | O(n²) | O(1) |
| HashSet | O(n) | O(n) |

---

# 🧠 Algorithm

1. Create an empty HashSet.
2. Traverse array.
3. If current element already exists

```
Duplicate Found
```

Otherwise

Store inside HashSet.
