# Move Zeroes - Interview Questions

## Q1. Which pattern is used?

```
Two Pointer
```

---

## Q2. Why not use sorting?

Sorting changes the relative order of elements.

---

## Q3. Time Complexity?

```
O(n)
```

---

## Q4. Space Complexity?

```
O(1)
```

---

## Q5. Why swap only when the current element is non-zero?

To preserve the order of non-zero elements while pushing zeros to the end.
